## 🚀 Cómo ejecutar el proyecto localmente

Dado que es un archivo estático, no requiere instalación de dependencias complejas. Puedes visualizarlo de las siguientes maneras:

**Opción 1: Visualización directa**
1. Clona este repositorio o descarga el archivo `CV_Joaquin_Najt_Basico.html`.
2. Haz doble clic en el archivo para abrirlo en tu navegador web predeterminado.

**Opción 2: Usando VS Code (Live Server)**
1. Abre la carpeta del proyecto en Visual Studio Code.
2. Instala la extensión "Live Server".
3. Haz clic derecho sobre el archivo HTML y selecciona `Open with Live Server`.

**Opción 3: Usando Python**
Si tienes Python instalado, puedes levantar un servidor local rápidamente:
1. Abre la terminal en la carpeta del proyecto.
2. Ejecuta el comando: `python -m http.server`
3. Abre tu navegador y ve a `http://localhost:8000`.
